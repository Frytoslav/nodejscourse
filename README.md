# Kurs Node.js i NestJS

To repozytorium zawiera kompletny plan kursu dla osoby, która zaczyna backend od podstaw i chce dojść do poziomu, na którym potrafi samodzielnie zbudować bardziej rozbudowane API w NestJS.

Start: [kurs-node-nestjs/README.md](kurs-node-nestjs/README.md)

Kurs prowadzi przez:

- fundamenty JavaScript potrzebne w backendzie,
- Node.js, npm, modularyzacje, asynchronicznosc i HTTP,
- pierwsze API bez frameworka i z Express,
- TypeScript jako codzienny jezyk backendu,
- NestJS: moduly, kontrolery, serwisy, DTO, walidacje, dependency injection,
- bazy danych, Prisma, relacje, migracje i transakcje,
- auth, role, bezpieczenstwo, testy, konfiguracje, logowanie,
- architekture wiekszego projektu, kolejki, cache, pliki, dokumentacje API i wdrozenie,
- projekt koncowy, ktory scala caly material.

Najwygodniej przechodzic modul po module i nie przeskakiwac zadan praktycznych. Teoria jest po to, zeby rozumiec decyzje, ale backendu uczysz sie dopiero wtedy, gdy samodzielnie doprowadzasz endpoint do dzialania.
